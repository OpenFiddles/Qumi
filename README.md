# Meet Qumi
Welcome to Qumi. The world's first self taught AI bot by OpenFiddles. Made of only Python's pre-installed modules (with exception for PySide 1) and custom made modules. You might be surprised at what it can do. So let's prepare for somethings. 

# How it Learns and Sore.
What you need to know is that the app will create a directory named \QumiData\ at whatever qumo.exe resides. Then inside the folder will create synapse.json, brain.db, and vocab.json. 

Synapse.json is nothing but a storage for weight of the QuNet (QuTensors's Neural Net) it holds.

Vocab.json is storage for it's vocabulary.

Brain.db is storage for all of each words it knows with word name, it's coordinates (x, y, z), and volts (interests).

# Standalone tools.
Qumi Explorer - A explorer that injects and pulls from brain.db and shows you points of nodes of each words on a 2D graph.

Qumi Editor - A editor that will only accept brain.db and it's hardcoded signature. But you can view and edit its data.
